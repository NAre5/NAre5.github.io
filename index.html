  1. The Three main diffrences between web1,web2 and web3

    1. web1 is a page only for one person that is only static or dynamic per person, but for web2 and web3 the pages can be more dynamic, where a person can change the site and it will effect how other person will see the web page on their device.
      also web3 refreshing more often than web2.
    2. from search in the web prespective: web1 only can have you search specific url while web2 enable search of webpage with search engine with set before tags. web3 enables more then that and enables semantic search by the web page content.
    3. in web2 and web3, you could use crawlers on the whole web.

  2. Medicines side effects Use Case

    3. we will use the example of the query: "Advil side effects"

    • web1: the user need to write specific url of a site talking about a site about Advil side effects
    • web2: the user can search in the search enguine- "Advil side effects" and get links to web pages that have tags of the words or pairs in the query.
    • web3: the user can search in the search enguine- "Advil side effects" and get links to web pages or specific part of web pages that connect well to the query topic.
